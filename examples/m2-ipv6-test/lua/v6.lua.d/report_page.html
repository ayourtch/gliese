The tests are sequential: A-only (start); A+AAAA, DNS server on IPv4; AAAA only, DNS server on IPv4; AAAA only, reply >1500 bytes, DNS server on IPv4; AAAA only, DNS server on IPv6 (thanks to Steve Bertrand for the record in the DNS)

The Opera browsers are completely excluded from the test results for the invocations that use "display:none" style - due to Opera not requesting the image which has the "display:none" style. Thanks to David Croft for noting that. Please use the "width:0px;height:0px" style to get the info about the Opera browsers too.

Important: If your referer can change (e.g. site accessible via "example.org" and "www.example.org", or something is passed via parameters in GET request, then to avoid the dispersion of the results, please supply the referer explicitly via "r=" parameter.
